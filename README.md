# plant-watering-system
Hello everyone, I am Aarav Yadav. Today, I am excited to share a repository that explains how to build a plant watering system using an ESP-32. You can incorporate this system into your garden and control it using your smartphone.

In this repository, you will find the circuit diagram, code, and other necessary resources. I will continue to update it regularly.
